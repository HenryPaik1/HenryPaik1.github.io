G00D Morning "Data"
2019-01-26
github
hexo

install Node.js

$ wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh

install hexo

$ npm install hexo-cli -g
$ hexo init blog # blog = <file name>
$ cd blog

set theme

$ git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus
$ vi _config.yml

setting deploy config.

$ npm install hexo-deployer-git --save
$ vi _config.yml

apply modification & run hexo server

$ hexo g
$ hexo s

sync github

$ hexo d

refernce: https://github.com/probberechts/hexo-theme-cactus https://hexo.io/docs/commands
